# US Accident Modeling
### DS 4420
### Evelyn Robert and Zoe Garbus

This project investigates factors that influence the severity of traffic accidents using machine learning and probabilistic modeling techniques. Using a large-scale U.S. traffic accident dataset containing environmental, roadway, and temporal variables, we analyze how different conditions contribute to accident severity. Two modeling approaches are implemented and compared: a neural network model to capture complex nonlinear relationships in the data, and a Bayesian probabilistic model to estimate interpretable relationships between accident factors and injury severity. The results of this analysis may provide insights that could support traffic safety research and decision-making.
