# US Accident Modeling
### DS 4420
### Evelyn Robert and Zoe Garbus

This project investigates factors that influence the severity of traffic accidents using machine learning and probabilistic modeling techniques. Using a large-scale U.S. traffic accident dataset containing environmental, roadway, and temporal variables, we analyze how different conditions contribute to accident severity as well as how accients fluctuate over time. Two modeling approaches are implemented: a time series model that decomposes daily accident frequency into trend, seasonal, and weekly components, and a Bayesian model that predicts accident severity (1-4) from environmental and situational features such as temperature, visibility, humidity, time of day, and weather conditions. The results of this analysis may provide insights that could support traffic safety research and decision-making.
